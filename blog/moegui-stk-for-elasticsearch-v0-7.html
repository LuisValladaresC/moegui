Moegui STK for ElasticSearch® release v0.7

We are proud to announce the last Moegui STK for ElasticSearch® release v0.7

This version uses ElasticSearch® and Kibana® 7.10.2

Changes Implemented

  • Stack SSM Parameters template moved to Galaxy role.
  • Stack Security template moved to Galaxy role.
  • Stack AutoScalingGroup template moved to Galaxy role.
  • Stack LoadBalancer template moved to Galaxy role.
  • Stack SSM State Manager template moved to Galaxy role.
  • Stack CodeSuite template moved to Galaxy role.
  • Updated Image to 7.10.2.

Where to get?

You can deploy or update this new version directly from our MarketPlace.
