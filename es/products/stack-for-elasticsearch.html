Moegui STK for ElasticSearch®

  • Information

  • Pricing

What is it?

It is an AWS CloudFormation set of a template to deploy an ElasticSearch® & Kibana® Stack with best practices configurations made by experts ready to be used.

Where to start?

We have prepared a complete Getting started to guide you over all the deployment processes identifying the most relevant concepts. This guide uses a full set of screenshots that captures where you can follow a smooth deployment process. Following inner links, you can go deeper for advanced features or modifications.

Stack Components

Our Stack deploys a curated Amazon AMI based on the official Ubuntu 18.04 LTS.

We add the following packages:

  • Docker Engine
  • AWS System Manager Agent
  • Python
  • Git
  • Ansible 2.9

We make many Security and Performance improvements needed for ElasticSearch®.
